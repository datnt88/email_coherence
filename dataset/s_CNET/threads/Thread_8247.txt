5530	5	314083	2896288	2896025	steven haninger	10/31/08 6:31 am	1.22543466e12	additional to consider with willy's reply	you can try to start the ps from the mb connector as well. if you have a dvm, you can see if the voltages are there. i've seen a number of power supplies that show evidence of an attempt to start but shut down immediately. watch the ps fan carefully when you try to power up the pc. if it twitches and stops, this means it attempted to start but shut down do to detecting a load problem. this is generally a bad power supply or something it's connected to. switching power supplies are tricky to troubleshoot as they also don't like to start when not connected to a load. if you can connect the ps to another mb and the ps starts ok, this usually rules it out as the problem. it doesn't rule it out as possibly being of insufficient power to operate what it's connected to. i'd remove all connections to the ps but the mb. this can also include ram and cpu if you wish. if the ps starts ok as evidenced by it's fan continuing to run when it hasn't before, this makes the mb suspect. it's also not uncommon to have standoffs under a mb causing short circuits if you have it in the case. you can check this by putting the mb on a table and just connecting things outside the box. good luck.
