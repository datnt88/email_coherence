you need to go to an ethernet hookup from a dsl modem, the cable input is not going to work! so what you need is a wireless router, not a cable gateway.
forgot to say that on a cable gateway the lan input is cable, dsl is ethernet you need ethernet from dsl modem to lan input on router, you do not have that on a cable gateway, there is a difference.
basically what you have using the gateway is a hardwire switch feeding off of the dsl modem but you do not have the proper signal input (lan/wlan) since the signal input for the gateway (cable) is totally different.
